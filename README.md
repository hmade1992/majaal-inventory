# نظام إدارة المخزون - شركة التميز

نظام شامل لإدارة مخزون البورسلين والرخام والمواد الصحية.

## المميزات

- **نظام المصادقة**: تسجيل دخول آمن باستخدام Supabase Auth
- **لوحة التحكم**: عرض إحصائيات شاملة للمخزون
- **الجرد اليومي**: تسجيل الجرد اليومي مع دعم الأرقام الكسرية
- **إدارة التوالف**: تسجيل وتتبع التوالف
- **التقارير**: تقارير تحليلية للأصناف الأكثر سحباً وتلفاً
- **إدارة المستخدمين**: إدارة المستخدمين والصلاحيات (للمشرفين فقط)
- **تنبيهات المخزون**: تنبيهات تلقائية عندما تقترب الأصناف من النفاد

## التقنيات المستخدمة

- HTML + CSS + JavaScript (بدون Frameworks)
- Supabase (قاعدة البيانات والمصادقة)
- Vite (أداة البناء)

## الإعداد والتشغيل

### 1. تثبيت المتطلبات

```bash
npm install
```

### 2. إعداد Supabase

1. أنشئ حساب على [Supabase](https://supabase.com)
2. أنشئ مشروع جديد
3. انسخ URL و Anon Key من لوحة تحكم Supabase
4. أنشئ ملف `.env` في جذر المشروع:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. تشغيل المشروع

```bash
npm run dev
```

### 4. بناء المشروع للإنتاج

```bash
npm run build
```

## قاعدة البيانات

تم إنشاء الجداول التالية:

### items (الأصناف)
- معلومات الأصناف مع الكميات الحالية
- دعم الأرقام الكسرية
- حد أدنى للتنبيه

### daily_inventory (الجرد اليومي)
- تسجيل الجرد اليومي
- حساب الفرق تلقائياً
- تحديث الكمية الحالية تلقائياً

### damages (التوالف)
- تسجيل التوالف مع الأسباب
- خصم الكمية من المخزون تلقائياً

### user_profiles (المستخدمون)
- معلومات المستخدمين والأدوار
- ثلاثة أدوار: مدير، مشرف، موظف

## الأدوار والصلاحيات

- **موظف**: الجرد اليومي، التوالف، عرض التقارير
- **مشرف**: جميع صلاحيات الموظف + إضافة وتعديل الأصناف
- **مدير**: جميع الصلاحيات + إدارة المستخدمين

## التنبيهات

- تنبيه تلقائي عندما تقل الكمية عن الحد الأدنى (افتراضياً 60)
- عرض الأصناف المنخفضة بلون تحذيري
- عداد للأصناف القريبة من النفاد
